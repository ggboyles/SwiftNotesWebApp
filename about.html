About SwiftNotes

SwiftNotes is a simple web application created in 2023 for easy note-taking. Built with HTML, PHP, and MySQL, SwiftNotes allows registered users to log in, create, and delete notes securely. With a clean and user-friendly interface, SwiftNotes is easily accessible for all tech users, from new to old.
